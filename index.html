Wifi Analyzer

A tool to inspect, analyze and return in depth information about your home or corporate wifi network.

Find Out More!

Coded by Cole Stasnel & Sadathe Toure

Section A

The Wifi Analyzer is a device made by Cole Stasney and Sadathe Toure for a week long Q3 project during their Galvanize's Web Development Immersive program. This device allows you to inspect, analyze and return in depth information about your home or corporate wifi network to determine the power, privacy setting, authentication and channel of you network.


We used a Raspberry Pi, a touchscreen monitor and a network card to configure this device. The application was written in JavaScript, Express.js, Node.js for the back end.
